# Sistema de Controle de Chamados

## Objetivo
Aplicação web para registrar e acompanhar chamados de suporte.  
Projeto da disciplina de Desenvolvimento de Software, com foco em funcionalidade completa (Front-end + Back-end + Banco de Dados).

---

## Arquitetura
- **Front-end**: HTML, CSS, JavaScript  
- **Back-end**: Node.js + Express  
- **Banco de Dados**: MySQL  

---

## Funcionalidades
- **AC1**: Criar e listar chamados  
- **AC2**: Atualizar status  
- **AC3**: Excluir chamado / filtro por status  
- **Prova Final**: Editar chamado + Diagrama de Casos de Uso
